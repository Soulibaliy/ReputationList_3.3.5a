# Reputation - Менеджер списков игроков для WoW 3.3.5a

Комплексный аддон для ведения черного списка, белого списка и заметок игроков с расширенными возможностями отслеживания.

##  Возможности

### Карточки игроков
Отображают полную информацию:
- GUID (уникальный идентификатор)
- Имя, расу, класс, уровень
- Гильдию
- Заметку
- Дату и автора записи

### Отслеживание GUID
Позволяет отслеживать игроков после смены имени/расы/фракции. Проверка происходит:
- В группе/рейде
- При взятии в цель / mouseover
- При начале обмена

### Интерфейс
- Локализация: RU и EN
- Два стиля: Classic и ElvUI
- Двойной интерфейс: Собственное окно + встроенный в окно "Общение"

### Импорт из других аддонов
Поддерживает импорт из других аддонов

### Управление игроками
- Добавление через ПКМ на игрока
- Добавление из чата
- Данные для всего аккаунта с разделением на реалмы (x1, x4, x100)

##  Типы списков

- Blacklist — Проблемные игроки
- Whitelist — Надежные игроки (танки, хилы, друзья, РЛ)
- Notelist — Нейтральные заметки (например: «часто AFK»)

##  Отслеживание и уведомления

- Авто-оповещение при входе в группу/рейд
- Уведомление о сообщениях и обмене
- Проверка участников в реальном времени
- Всплывающие карточки
- Подсказки в tooltip'ах
- Звуки уведомлений
- Цветовое выделение в LFG каналах
- Интерактивные метки на вкладках Гильдия/Рейд

## Интеграция с встроенным игнор-листом

- Прямое добавление в игнор
- Отображение статуса
- Быстрое снятие игнора
- Счетчик занятых слотов

## Блокировка от игроков из Blacklist

Блокировка:
- Приглашений в группу
- Обмена
- Сообщений в ПМ
- Кнопка для быстрого добавления в BL и исключения из группы/рейда (только для лидера)

## Копирование и перенос данных

### Способ 1: Программа объединения (рекомендуется)
Используйте программу для объединения списков на нескольких аккаунтах.

### Способ 2: Ручной экспорт/импорт
Скопируйте файл вручную:
```
/WTF/Account/Имя_аккаунта/SavedVariables/reputation.lua

```
Либо используйте внутриигровой экспорт/импорт

## Установка

1. Скачайте последнюю версию
2. Распакуйте папку `Reputation` в директорию аддонов WoW:
```
   World of Warcraft/Interface/AddOns/
```
3. Перезапустите WoW
4. Используйте `/rlnew`, `/rlelvui` или значок на мини-карте  для открытия интерфейса. 
Либо используйте аддон во встроенном окне Общение.

## Использование программы объединения

Что делает:
- Проверяет наличие файла `SavedVariables\reputation.lua` во всех аккаунтах
- Объединяет списки из файлов в единый
- При дублировании записей берет более свежую
- Записывает готовый файл в папки всех аккаунтов

Как работать:
1. Закройте WoW клиент (обязательно!)
2. Запустите программу
3. Укажите путь до папки `\WTF\Account`
4. Нажмите "Объединить данные"
